# 📘 Vocab App

A vocabulary learning platform to help users build their word bank daily, track learned words, and improve English with powerful features.

---

## 🚀 Features

- 🔐 **Authentication**
  - Sign in & sign out using Supabase Auth
  - Middleware-based protected routes
  - Auth callback 

- 📋 **Dashboard**
  - Daily curated English words with meanings, examples, synonyms, etc.
  - Tracks reading and progress using Supabase DB

- 📘 **Learned Words**
  - View and review words you've marked as learned
  - Track vocabulary growth over time

- 🌐 **Prisma + Supabase**
  - Serverless APIs 
  - Supabase as backend database with Prisma

- 🎨 **UI/UX**
  - Built with Next.js App Router (`/app`)
  - Tailwind CSS + ShadCN UI components
  - Responsive sidebar and loading indicators

---

## 🛠️ Tech Stack

- **Frontend**: Next.js 14+ (App Router), React, Tailwind CSS, ShadCN UI
- **Backend**: Prisma, Supabase (Auth + Database)
- **Auth**: Supabase JWT tokens, cookie-based session
- **Dev Tools**: TypeScript, ESLint, Prettier, GitHub Actions
